# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and give actionable feedback.
- Focus discussions on technical outcomes.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Publishing private information without permission
- Disruptive or intentionally hostile behavior

## Enforcement

Maintainers may remove, edit, or reject contributions and comments that violate this code of conduct.

## Scope

This policy applies in project spaces, issues, pull requests, and community channels related to this repository.
